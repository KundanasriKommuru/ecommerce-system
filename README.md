# 🛒 E-Commerce Order System

## 📌 Project Description
This is a console-based E-Commerce system built using Python and MySQL.
It supports user authentication, product management, order processing, and analytics.

---

## 🚀 Features
- User Registration & Login
- Product Management
- Order Placement
- Order History
- Summary Dashboard

---

## 🛠️ Tech Stack
- Python
- MySQL
- Git & GitHub

---

## 🗄️ Database Tables
- users
- products
- orders
